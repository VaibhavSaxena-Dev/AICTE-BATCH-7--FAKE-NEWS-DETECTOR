# Fake News Detector

A full-stack application to detect fake news with MongoDB Atlas backend and React frontend.

The backend will run on http://localhost:5000


The frontend will run on http://localhost:5173

## Features

- User authentication (signup/login)
- Article validation checking
- History tracking for logged-in users
- MongoDB Atlas database
- JWT-based authentication

In the backend folder 

MONGODB_URI=

JWT_SECRET=

PORT=5000

ML_SERVICE_URL=

To connect to the data base which is mongo db atlas , setting up the token key JWT_SECRET


In the mlService folder 


GEMINI_API_KEY=


to connect to the gemini key
